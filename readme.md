

项目名称：多源新闻舆情智能 Agent
项目定位：面向新闻信息流的轻量 NLP/ML 工程项目，完成“数据抓取→文本清洗→主题聚合→情绪分析→摘要生成→自动推送”闭环。
技术栈： Python / Requests / Flask / YAML / Webhook / 规则+词典NLP / GitHub Actions / Docker

项目描述：
- 设计并实现多源新闻采集与结构化处理流程，接入 11 个平台，单次处理 250+ 条热点数据，支持失败重试与统一数据落盘。
- 实现关键词级 NLP 分析管线：标题清洗、归一化去重、主题词提取、领域归类（政治/经济/科技三选一）与情绪倾向判断（正/中/负）。
- 重构消息生成策略：将传统“趋势/预测”改为 120字内核心摘要 + 领域影响解读 + 情绪分布图示，显著提升信息密度与可读性。
- 完成推理效率优化：设置单关键词最多分析 15 条新闻，结合内容长度控制与降级发送策略，保障企业微信机器人稳定触达。
- 新增“当前时段单次分析”命令入口，实现仅抓取当前时段并实时推送，适配高时效场景下的轻量在线推理需求。
